# 夜幕疾影 NightTrace

《夜幕疾影》是一款面向 Android 手机的竖屏三车道跑酷游戏。玩家驾驶追影摩托驶入夜色高架，在不断加速的道路上观察障碍、切换车道、跳跃和滑铲，一边躲避施工物、油渍、激光门与移动车辆，一边收集金币、钻石和临时强化道具，向更远距离和更高分数发起挑战。

项目使用 Android 原生 Java 与自定义 `View` 实现，围绕课程设计规模做成了一个完整可运行的游戏闭环：艺术标题开始页、开局倒计时过场、触控操作、障碍生成、道具反馈、生命值、暂停菜单、失败结算和最高分保存都已经接入。画面采用夜幕城市高架主题，通过滚动赛道、透视车道、车道光带、粒子、震屏和遮挡反馈强化速度感。

## 游戏特色

- 三车道无尽跑酷：规则清晰，开局即可通过滑动操作进入节奏。
- 多种障碍反馈：雪糕桶消耗生命，油渍触发打滑和视野遮挡，激光门与移动车辆要求及时应对。
- 局内强化系统：护盾、磁吸和冲刺分别提供容错、收集效率和高速得分体验。
- 高分挑战目标：距离推进带来阶段提升与速度压力，历史最高分会在本地保存。
- 开场表现完整：艺术字标题进入开始页，点击后先倒数，再播放 `LET'S GO` 过场进入游戏。
- 原生 Android 实现：核心循环、碰撞、生成、输入和绘制集中在自定义 View 中，便于阅读与课程展示。

## 玩法

- 触屏：点击开始；左右滑动换道；上滑跳跃；下滑滑铲。
- 调试键盘：A/D 或方向键换道；W/上/空格跳跃；S/下/J 滑铲；P 暂停；Enter 重开。
- 生命：初始 3 点。雪糕桶扣 1 点生命并减速；油渍打滑、换道并短暂遮挡视野。
- 躲避：跳过雪糕桶，滑铲穿过激光门，移动车辆靠换道避开。
- 收集：金币是常见奖励，钻石是低概率高分奖励。
- 道具：护盾挡车辆或激光门；磁吸吸金币；冲刺短暂无敌并加快得分。

## 游戏流程

1. 在开始页点击开始，倒计时与 `LET'S GO` 过场结束后摩托驶入夜幕高架。
2. 观察前方车道，通过换道、跳跃和滑铲避开对应障碍。
3. 收集金币、钻石与强化道具，在更高速度阶段维持节奏。
4. 生命耗尽或撞上未被抵消的致命障碍后结算本局分数，并继续挑战最高纪录。

## 代码入口

- `MainActivity`：负责开始界面、暂停菜单、最高分读写和状态联动。
- `NightTraceView`：负责游戏循环、输入处理、对象生成、碰撞判定、HUD 与画面绘制。
- `app/src/main/res`：保存布局、主题、游戏位图素材和应用文案。
- `课程设计报告/夜幕疾影课程设计报告.pdf`：随仓库同步的课程设计报告 PDF。

## 运行方式

1. 使用 Android Studio 打开本目录。
2. 等待 Gradle 同步完成。
3. 选择 Android 模拟器或真机运行 `app`。

命令行也可以先执行一次调试构建：

```bash
./gradlew :app:assembleDebug
```

构建成功后，可将 `app/build/outputs/apk/debug/app-debug.apk` 传到 Android 手机安装试玩。工程本地根目录也可放置便于提交作业的安装包副本，但 APK 默认不纳入 Git 版本管理。
